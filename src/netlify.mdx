# mdx-deck

```bash
# start the dev server
mdx-deck [path-to-file]

# build static HTML page with JS bundle
mdx-deck build [path-to-file] -d [dest-folder]

# Export PDF
mdx-deck pdf [path-to-file] --out-file [dest-path-to-file]
```

---

# Mdx-deck 与 Netlify 结合

- Mdx-deck 生成 slides
- Netlify 执行构建并部署

---

# Mdx-deck + Netlify

```bash
# NPM scripts
"scripts": {
  "build": "mdx-deck build [path-to-file] -d [dest-folder]",
}

# Netlify
npm run build
```

---

# 多个 slides

```bash
# Build
mdx-deck build src/[filename].mdx -d dist/[filename]/

# NPM scripts
"scripts": {
  "build": "node build/index.js",
}

# Netlify
npm run build
```
