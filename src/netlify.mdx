- Mdx-deck 生成 slides
- Netlify 执行构建并部署

---

# Mdx-deck + Netlify

```bash
# NPM scripts
"scripts": {
  "build": "mdx-deck build [path-to-file] -d [dest-folder]",
}

# Netlify
npm run build
```

---

# 多个 slides

```bash
# Build
mdx-deck build src/[filename].mdx -d dist/[filename]/

# NPM scripts
"scripts": {
  "build": "node build/index.js",
}

# Netlify
npm run build
```
